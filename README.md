# Roberta Mazzone Style Skill

生成意意式慢旅行图片Skill。

它提炼了 Roberta Mazzone / @_lagiuditta 启发的意式慢旅行摄影语言，用于给你的照片增加滤镜。

摄影师账号：https://www.instagram.com/_lagiuditta/

<img width="1892" height="1134" alt="image" src="https://github.com/user-attachments/assets/e35a2634-382a-41f3-b2d9-1f5b216e3912" />


这个 Skill 的目标不是复制某一张具体作品，而是把这种风格拆解成可执行的视觉方法：场景、构图、光线、色彩、材质、人物姿态和后期方向。
